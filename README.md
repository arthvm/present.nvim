# `present.nvim`

A simples plugin for presenting markdown files

# Features

We can execute code in lua blocks!

```lua
print("Hello world!")
```

# Features (Part 2)

We can execute code in javascript blocks too!

```javascript
console.log({myField: true, other: false})
```

# Features (Part 3)

Even python!

```python
print("Blablabla python")
```
